# AgendaDiscordBot

This is a Discord bot that allows users to create agendas for meetings. It is written in Python and uses the discord.py library.

## Installation

To install the bot, you need to have Python 3.6 or higher installed. You also need to have the discord.py library installed. You can install it by running `pip install discord.py` in your terminal.

## Usage

```bash
python setup.py
```
